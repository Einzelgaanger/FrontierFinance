# Documentation Directory

This directory contains all project documentation files.

## Contents

### Markdown Files (.md)
- Project documentation
- Setup and installation guides
- Migration guides
- Handover reports
- Technical documentation
- API documentation
- Data import/export guides

### Text Files (.txt)
- Database schema exports
- Configuration notes
- Log files

### LaTeX Files (.tex)
- Formal documentation
- Academic papers
- Deployment reports

## Organization

All documentation files are stored in this directory for easy access and reference. Use the file names to identify the topic or purpose of each document.
