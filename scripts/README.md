# Scripts Directory

This directory contains all utility scripts organized by purpose.

## Directory Structure

### `migrations/`
Database migration and schema update scripts.
- Migration execution scripts
- Data migration utilities
- Schema migration helpers

### `imports/`
Data import scripts for surveys, users, and other data.
- Excel import scripts
- Survey data import scripts
- Bulk data import utilities
- Re-import scripts

### `utilities/`
General utility scripts for various tasks.
- File manipulation scripts
- Column mapping generators
- User account creation utilities
- Email utilities
- Component replacement scripts
- Password update scripts

### `tests/`
Testing and debugging scripts.
- Connection test scripts
- Debug utilities
- Test HTML files

### `config/`
Configuration and dependency files.
- Python requirements files
- Migration requirements
- Configuration templates

## Usage

Scripts are organized by function. Check the appropriate subdirectory for the script you need.
